# 🔋 프로젝트 제목: 배터리 SOH 진단기(20252830 우승제)

## 📌 개요
전자공학 전공 지식을 활용하여 개발한 **산업용 배터리 수명(SOH) 예측 웹 어플리케이션**입니다.
고가의 전문 계측 장비 없이도 배터리의 화학적 특성과 사용 이력(사이클, 온도, 방전 심도 등) 데이터를 기반으로 잔존 수명을 정밀하게 진단할 수 있도록 설계되었습니다. 현장 업무 효율성을 위해 모바일 환경에 최적화되어 있습니다.

## ⚙️ 주요 기능
- **정밀 수명 예측**: 단순 횟수 차감이 아닌, NREL(미국 국립재생에너지연구소)의 반경험적 모델을 적용하여 **캘린더 노화(시간)**와 **사이클 노화(사용)**를 분리 계산
- **다기종 지원**: 리튬이온(NMC), 인산철(LFP), 납축전지(Lead-Acid) 등 화학 조성별 고유 물성 데이터(DB) 반영
- **고급 분석 옵션**: 방전 심도(DOD) 및 충전 속도(C-Rate)에 따른 가속 열화 시뮬레이션
- **현장 리포팅**: 진단 결과를 자동 요약하여 카카오톡 등으로 즉시 공유할 수 있는 **'원클릭 리포트 복사'** 기능

## 🧪 적용된 공학적 원리 (Engineering Logic)
- **아레니우스 방정식 (Arrhenius Eq)**: 온도 상승에 따른 화학 반응 속도(열화) 가속화 구현
- **Square-root Time Law ($\sqrt{t}$)**: 시간 경과에 따른 초기 SEI 피막 형성 속도의 비선형성 반영
- **Power Law**: 방전 심도(DOD) 깊이에 따른 스트레스 지수함수 적용

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Distribution**: GitHub Pages
- **Environment**: Mobile First Design (PWA-like experience)

## 🚀 실행 및 사용 방법
1. **[https://milgaroo05.github.io/battery-soh-estimator/]** 에 접속합니다.
2. (선택) '홈 화면에 추가'를 통해 앱처럼 설치합니다.
3. 배터리 종류, 용량, 사용 기간, 싸이클 횟수 등을 입력합니다.
4. **[정밀 진단 실행]** 버튼을 누르면 SOH(수명)와 상세 분석 그래프가 표시됩니다.
5. **[리포트 복사]** 버튼을 눌러 진단서를 공유합니다.
